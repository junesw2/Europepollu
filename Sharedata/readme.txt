Unclean Air Exposure Time Dataset
Overview
This dataset provides information on the exposure time to various air pollutants (PM2.5, PM10, NO2, O3) in different locations across multiple years. It includes the following columns:

ID: Unique identifier for each entry.
year: Year of the recorded data.
location: Location code or identifier.
pm25v0_expo_day: Exposure time to PM2.5.
pm10v0_expo_day: Exposure time to PM10.
no2pred_expo_day: Exposure time to NO2.
o3_8hpred_expo_day: Exposure time to O3.

Data Structure
ID: Integer
year: Integer
location: String
pm25v0_expo_day: Float
pm10v0_expo_day: Float
no2pred_expo_day: Float
o3_8hpred_expo_day: Float
